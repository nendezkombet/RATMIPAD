- Untuk "Ratmi7layer" jika menggunakan RGB led direkomendasikan acrylic warna transparent/clear, sedangkan "Ratmi7topbottom" warna acrylic bebas.
Untuk ketebalan acrylic keseluruhan = 3mm

- Sebelum mengirim file laser cutting di atas ke jasa acrylic laser cutting harap me-rename file dengan nama warna yang dipilih untuk memudahkan pihak jasa laser cutting dalam proses cutting.
Ex. Ratmi7layer --> 3mmclearfrosted 
    Ratmi7topbottom --> 3mmblackfrosted

- Untuk assembly dibutuhkan :
	- 4x 10mm M2 "MALE TO FEMALE" brass standoff untuk bottom layer
	- 4x 10mm M2 "FEMALE TO FEMALE" brass standoff untuk top layer
	- 8x 5mm M2 screw untuk bottom dan top layer
